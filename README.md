## 🔄 同步规则

**仓库 = 工作区。** 不需要 sync.sh，不需要中间目录，不需要额外脚本。

- **读**：`cat 文件名` 就行，文件就在工作区里
- **写**：直接编辑文件，然后 `git add -A && git commit -m "msg" && git push`
- **拉**：`git pull origin master`

就这么简单。
